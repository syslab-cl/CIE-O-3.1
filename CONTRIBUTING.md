# Guía de Contribución

Gracias por tu interés en contribuir al proyecto CIE-O 3.1.

## Cómo contribuir
1. Haz un fork del repositorio
2. Crea una rama nueva
3. Envía un Pull Request claro y documentado

## Estándares de commits
Usar formato:
- feat: nueva funcionalidad
- fix: corrección
- docs: documentación
